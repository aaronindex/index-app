# checklist.md — INDEX Execution

Operate this weekly.  
If a gate is not met, pause and correct before proceeding.

---

## WEEK 1 — Distribution Readiness

- [ ] Onboarding tour behaves correctly on first login
- [ ] Empty states + post-import copy reflect reduction → focus → carry forward
- [ ] Activation events firing reliably:
  - [ ] import_completed
  - [ ] insights_extracted
  - [ ] task_created
  - [ ] ask_index_answered (has_answer = true)
- [ ] New user can plausibly reach first value in one session

**Gate**
- [ ] User reaches first value without explanation

---

## WEEK 2 — First External Users (10–20)

- [ ] Invite 10–20 high-signal users
- [ ] Maintain no-coaching default
- [ ] Observe and log:
  - [ ] stall points
  - [ ] value moments
  - [ ] confusing language

**Gate**
- [ ] ≥40–50% reach first value
- [ ] ≥3 users describe same core benefit

---

## WEEK 3 — Expanded Observation (25–50)

- [ ] Expand access beyond inner circle
- [ ] Maintain no-coaching default
- [ ] Segment users:
  - [ ] did not reach value
  - [ ] reached value once
  - [ ] returned for second session

**Gate**
- [ ] Users explain INDEX’s difference without borrowing founder language

---

## WEEK 4 — Activation Clarity

- [ ] Review activation funnel
- [ ] Identify single largest drop-off
- [ ] Decide: proceed or tighten onboarding

**Gate**
- [ ] Clear activation narrative
- [ ] One dominant friction point identified

---

## WEEKS 5–8 — Repeatable Activation

### Weekly
- [ ] Publish one real INDEX artifact
- [ ] Personally invite new users
- [ ] Run one small-group exposure
- [ ] Fix one onboarding or clarity issue max

**Weekly Gates**
- [ ] 10–20 new users
- [ ] ≥40% reach first value
- [ ] ≥20% return for second session

---

## WEEKS 9–12 — Monetization Signal

- [ ] Introduce light continuity-based paid boundary
- [ ] Observe upgrade behavior
- [ ] Measure time from first value → upgrade attempt

**Gate**
- [ ] ≥2–5% of activated users attempt upgrade
- [ ] Clear “why I paid” language emerges

---

## Do Not Advance Until All True

- [ ] Activation is repeatable
- [ ] Monetization signal exists
- [ ] Growth mechanics explainable in one paragraph
